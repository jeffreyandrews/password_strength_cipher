# password_strength_cipher
Combination of my password strength checker and caesar cipher tool to make a program that creates strong passwords and encodes them. This program requires Python 3 and should be run using password_strength_cipher.py in the terminal. Along with combining the two codes, I made each block of code into its own function to enhance  readability and make future to edits easier. I also added password confirmation to ensure the user enters their intended password, and I fixed a few bugs and made some quality-of-life improvements to the code in the process. This project really helped me understand the basics of how to keep important data safe.
Key features:
- Password validation with strength rating
- Caesar cipher encoding
- Password confirmation on entry
- 5-attempt lockout before account deactivation
